# Proyecto de Sistemas Operativos

Proyecto desarrollado en C orientado a la simulación y gestión de procesos, utilizando estructuras típicas de sistemas operativos como colas, tablas de procesos y bloques de control.

## Descripción

Este proyecto implementa conceptos fundamentales de sistemas operativos mediante estructuras y módulos que permiten representar procesos, su organización y su ejecución.

## Componentes principales

- **Bloque de Control de Proceso**
- **Cola de procesos**
- **Tabla de procesos**
- **Gestión del sistema**
- **Registro de ejecución**
